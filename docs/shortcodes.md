# Shortcodes

## Available Shortcodes

### `[bdss_my_subscription]`
Displays the logged-in user's subscription-related information.

**Use case:**  
Add this shortcode to a page where users can view their subscription details.

### `[bdss_subscription_status]`
Displays the current subscription status for the logged-in user.

**Use case:**  
Useful for account, membership, or dashboard pages.

### `[bdss_protect]`
Protects content based on plan or access rules.

**Example:**
```shortcode
[bdss_protect plan="premium"]
This content is only visible to authorized users.
[/bdss_protect]
